# Fyurr(LARAVEL)

A clone of fyuur(Python) project.



# Tech Dependencies
Tech used include:
- **PHP/Laravel** server language and framework
- **MySQL** database

## Run Locally

Clone the project

```bash
  git clone https://github.com/carol100/fyurr-laravel.git
```

Go to the project directory

```bash
  cd fyurr-laravel
```

Install dependencies

```bash
  composer install
```

Start the server

```bash
  php artisan serve
```
Run vite manifest

```bash
  npm install && npm run dev
```
